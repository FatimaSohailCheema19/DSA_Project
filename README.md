# 🎉 Event Management System (DSA Project)

## 📌 Overview
This project is an **Event Management System** developed using Java, based on Data Structures and Algorithms (DSA) concepts.

The system allows users to create, manage, and organize events efficiently. It demonstrates how data structures can be used to store, search, and manage event-related data in a structured way.

## 🎯 Purpose
The main goal of this project is to apply DSA concepts in a real-world scenario by building a system that handles event data such as event details, users, and bookings.


## 🧱 Project Structure
The project includes multiple classes such as:

- `Main.java` → Entry point of the system  
- `Event.java` → Represents event details  
- `User.java` → Handles user information  
- `Booking.java` → Manages event bookings  
- Other supporting classes (if any)

## 🧠 DSA Concepts Used
- Arrays / ArrayLists for data storage  
- Searching algorithms (to find events/users)  
- Sorting (if applied)  
- Data traversal techniques  
- Efficient data management  

## ⚙️ How to Run

1. Open terminal / command prompt  
2. Navigate to project folder  
3. Compile files:

```bash
javac *.java
```

4. Run the main class:

```bash
java Main
```

## ✨ Features
- Add and manage events  
- View event details  
- Book events  
- Store and retrieve records efficiently
